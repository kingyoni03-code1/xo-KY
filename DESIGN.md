# Cyber-Vibrant Design System

## 1. Visual Narrative
This design system represents a fusion of high-performance technical aesthetics and high-contrast digital vibrancy. Set against a deep, near-black void, the interface utilizes electric cyan and neon purples to create a sense of depth and energy. The transition to moderate roundedness and standard spacing creates a professional yet futuristic user experience that balances readability with a cutting-edge "cyber" personality.

## 2. Typography
The typography strategy prioritizes clarity and a technical edge.
- **Headlines & Labels**: Space Grotesk. The geometric nature of this font provides a monospace-inspired structure that feels calculated and modern.
- **Body Text**: Manrope. This font ensures high legibility for long-form content, bridging the gap between technical headlines and human-centric readability.

## 3. Color Palette
The palette is built on a "Vibrant Dark" foundation, emphasizing high-saturation accents against a dense neutral base.
- **Primary (#00E5FF)**: A piercing cyan used for primary actions, progress indicators, and focal points.
- **Secondary (#FF00FF)**: A vivid magenta for supporting UI elements and interactive feedback.
- **Tertiary (#BC13FE)**: An electric purple used for decorative highlights and distinct categorization.
- **Neutral (#05050A)**: A deep, midnight-black foundation that allows the vibrant accents to "glow" without causing visual fatigue.

## 4. UI Geometry & Spatial Rhythm
The system has been adjusted to provide a more balanced and accessible physical feel.
- **Roundedness**: Level 2 (Moderate). Moving away from sharp technical corners to a more standard rounded profile provides a contemporary, software-driven feel that is approachable yet precise.
- **Spacing**: Level 2 (Normal). The layout follows a standard spatial scale that provides ample room for elements to breathe without feeling overly sparse, maintaining a functional density suitable for complex dashboards and applications.

## 5. Components
- **Buttons**: High-contrast fills using the primary cyan, featuring moderate corner radii for a balanced look.
- **Surfaces**: Layered elevations using slight variations of the neutral base, accented with secondary or tertiary borders to define structure in the dark mode environment.
- **Data Visuals**: Utilizing the vibrant palette to ensure critical information stands out clearly against the dark background.
